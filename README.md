# GIF SONIC

- Trending GIFs
- Filtered GIFs
- Infinite Scroll
- Multiple Themes
- Play/pause GIFs
- Download GIFs
- Share GIFs
- Cross platform support

`Screens/main.js`
Includes main screen of the application

# Installation

First run `npm install` to download all dependencies.

Then simply run `npx expo start`

Note that this build is developed on expo sdk 50.
